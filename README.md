# Honor-of-Kings-Lingbao
王者荣耀灵宝售价通知
🚀 **零延迟、轻量级**的“小马糕”信息监控工具。支持自定义价格区间筛选，发现符合条件的商品后，通过 **Bark (iOS)** 或 **ntfy (Android/iOS)** 毫秒级推送到手机，并自动提取兑换码到剪贴板。

---

## ✨ 核心功能

- **⚡ 极速监控**：1秒轮询（可调），新消息发现即处理，无延迟。
- **🔍 智能解析**：自动识别文本中的价格和兑换码（支持多种格式）。
- **🛡️ 智能去重**：基于文本MD5和兑换码双重去重，拒绝刷屏骚扰。
- **📱 双通道推送**：
    - **Bark**：iOS专属，支持自动复制兑换码到剪贴板。
    - **ntfy**：Android/iOS/Web通用，支持点击复制。
- **⚙️ 高度自定义**：支持设置价格区间（如 800-899元）、监控关键词。
